# Gramedia Clone

## Deploy link

- `https://gramedia.vercel.app/`

## Ringkasan

App ini merupakan sebuah web aplikasi dengan tema eCommerce, yang dibuat dengan tujuan pembelajaran menggunakan:

- Typescript
- Framework NEXT.js
- MongoDB
- React-infinite-scroll-component
- Tailwind CSS
